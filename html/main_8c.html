void getValuesFromDB ( DATA *  dat)

This sets variables used when minimising the energy function.

Depending of the value of dat->natom this sets the variables dat->E_steepD, which may be used as a threshold for starting Steepest Descent minimisation, and dat->E_expected which is the energy of the best minimum.

Parameters
datis a structure containing control parameters common to all simulations.

This function starts a standard Metropolis Monte Carlo simulation.

This function is first in charge of opening all the output (coordinates, trajectory and energy) files.
Then the function #make_MC_moves starting the simulation is called.
In the end it prints results, close the files and goes back to the function main.
